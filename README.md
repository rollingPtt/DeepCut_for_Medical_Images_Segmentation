# DeepCut_for_Medical_Images_Segmentation

by 1ch 2025.11.7 for "Machine Learning and Medical Image Analysis"（机器学习及医学图像分析）

其中 `segment.py`, `util.py`, `features_extract.py` 三个文件相比于原始代码有所修改；

`try.py` 用于测试；

`voctest.py` 用于对 VOC07/12 数据集进行相关实验；

`HAM10000` (skin-cancer) 数据集可以从[这里](https://github.com/VahidMohammadZadehEivaghi/Skin-Cancer-Detection)下载；

`synapse` 数据集可以从[这里](https://gitcode.com/open-source-toolkit/df657/blob/main/Synapsedata.rar)下载.

此外，`results(part).zip` 中提供了部分结果。
